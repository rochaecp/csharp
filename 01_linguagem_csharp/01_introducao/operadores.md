## Operadores Aritméticos  

| Operador  | Nome do Operador      | Exemplo   |
| :---:     | :---:                 | :---:     |
| ```+```   | Adição                |           |
| ```-```   | Subtração             |           |
| ```*```   | Multiplicação         |           |
| ```/```   | Divisão               |           |
| ```%```   | Módulo                |           |
| ```++```  | Pré ou pós incremento |           |
| ```--```  | Pré ou pós decremento |           |
