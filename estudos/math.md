# C# - Math

> Namespace: System - ``` using System; ```

- Math.Max

    ```csharp
    myInt = Math.Max(2, 5);
    ```

- Math.Min

    ```csharp
    myInt = Math.Min(2, 5);    
    ```
    
- Math.Sqrt

    ```csharp
    myDouble = Math.Sqrt(64);    
    ```
    
- Math.Abs

    ```csharp
    myInt = Math.Abs(-10);   
    ```
    
- Math.Round

    ```csharp
    myDouble = Math.Round(9.99); // 10        
    ```
