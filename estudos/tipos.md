# C# - Tipos

## Tipos por Valor

- ``` byte ```  
    - 1 byte unsigned 0 to 255
- ``` sbyte ```  
    - 8 byte signed -127 to 128
- ``` short ```
    - 2 bytes signed -32768 to 32767
- ``` ushort ``` 
    - 2 bytes unsigned 0 to 65535
- ``` int ```
    - 4 bytes signed -2147483648 to 147483647
- ``` uint ``` 
    - 4 bytes unsigned 0 to 4294967295
- ``` long ``` 
    - 8 bytes signed
- ``` ulong ``` 
    - 8 bytes unsigned - ex.: 10L
- ``` float ``` 
    - 4 bytes 7 digits - ex.: 10F ou 35e3F
- ``` double ``` 
    - 8 bytes 15 digits - ex.: 10D ou 12E4D
- ``` decimal ``` 
    - 16 bytes 28 digits
- ``` char ``` 
    - 2 bytes - unicode
- ``` bool ``` 
    - 1 bit ex.: true, false
- ``` enum ```
- ``` struct ```
- ``` nullable ```

## Tipos por Referência

- ``` Class ```
- ``` Object ```
- ``` Array ```
- ``` Interface ```
- ``` Delegate ```

## Constantes

- ``` const``` 
    - ex.: ``` const int myConst = 10; ```


## Operadores

- ``` sizeof() ```
    - ex.: ``` int tamanho = sizeof(byte) ```
