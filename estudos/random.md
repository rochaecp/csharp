# C# - Random

- Métodos

    - Construtor

        ~~~csharp
        Random myRandom = new Random();        
        ~~~
    
    - Next()

        ~~~csharp
        myInt = myRandom.Next(10);
            // gera aleatorio de 0 a 9

        myInt = myRandom.Next(1, 10);
            // gera aleatorio de 1 a 10
        ~~~
