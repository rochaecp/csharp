# C# - Tuplas

## Criação

### Int, String

~~~csharp
(int, string) myTuple = (10, "Maurício");
~~~

## Acesso   

### Int, String

~~~csharp
myInt = myTuple.Item1;
myString = myTuple.Item2;
~~~    

## Modificação      

### Int, String

~~~csharp
myTuple.Item1 = 1;
myTuple.Item2 = "Maria";
~~~
