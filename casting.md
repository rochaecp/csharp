# C#

## Casting

### Casting Implícito

- Ocorre automaticamente.
- Converte um "tipo menor" em um "tipo maior":
    - char -> int 
    - int -> long
    - long -> float
    - float -> double

### Casting Explícito

- Ocorre de modo manual.
- Converte um "tipo maior" em um "tipo menor":
    - double -> float 
    - float -> long 
    - long -> int -
    - int -> char
